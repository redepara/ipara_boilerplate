Configurações Gerais

Requer HTML5 doctype

Os componentes do iPará foram desenvolvidos baseados no HTML5 Boilterplate. São de fácil utilização e customização. Alguns componentes são baseados no Bootstrap API. Para o uso dos mesmo é necessário a inclusão dos arquivos da API(estão inclusos com a API do iPará).

Arquivo config.js

Neste arquivos contém as opções de configuração do Site Conectado. 

var conf = {
	usuario : "deborarealeimoveis",
	qtdeDestaques : 7,
	anunciosPorPagina : 20
};

Destaques

Carousel

A API do iPará irá carregar a quantidade de destaques definidos no config.js e renderizar como um Carousel

Basta inserir o seguite código html na página desejada 

<div class="iparaCarousel" class="carousel slide"></div>

Listar Anúncios

A API do iPará irá carregar os anúncios do usuário e faz uma paginação de acordo com o parametro anunciosPorPagina definido no arquivo config.js

Para inserir listagem com filtro

Basta inserir o seguite código html na página desejada: 

<div title="filtro" class="filtroListar"></div>
<div title="listar" class="iparaListar"></div>

Para inserir apenas o filtro e a listagem acontecer em outra página

Utilize o seguinte html:

<div title="filtro" class="filtroListar post"></div>

Ao utilizar a classe post o botão filtrar irá redirecionar para a página listar.html



Detalhes de Anúncio

Para detalhar um anúncio utilize a página detalhes.html esta sempre vem com o parametro o anúncio a ser detalhado
Desta forma: detalhes.html?id=1234

A página deve conter o seguinte código html:

<div class="iparaDetalhes"></div>

Diversos componentes podem fazer parte da página de detalhes

Para carregar as fotos do anúncio utiliza-se o seguinte código:

<div class="fotosAnuncio" data-maxwidth="300" data-maxheight="300"></div>
